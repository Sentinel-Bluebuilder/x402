How It Works

Register. Pay. Connect.

Three steps from zero to private internet. The entire flow takes under 60 seconds.

1

Agent registers

Agent creates a Sentinel wallet (offline, free) and registers its address with our API. Gets back an agentId. Sentinel address never appears on Base or Solana.

Agent
2

Agent pays USDC

Calls contract.pay(agentId, 30) on Base. USDC transfers to operator wallet. VpnPayment event emitted on-chain. That's the only transaction the agent sends.

Base / Solana
3

Backend provisions access

Our backend sees the payment event, resolves agentId to the Sentinel address, and sends one atomic transaction: MsgShareSubscription + MsgGrantAllowance. Agent now has access and pays zero gas on Sentinel.

Automatic
4

Agent connects to VPN

Agent starts a session on Sentinel (zero gas, fee granted), handshakes directly with a VPN node, and establishes an encrypted tunnel. We never see the handshake, the tunnel credentials, or the traffic.

Agent ↔ Node

Connect on Base
import { connect } from 'x402-connect';

const vpn = await connect({
  payment: {
    chain: 'base',
    walletKey: process.env.EVM_PRIVATE_KEY,
    days: 30,
  },
  country: 'US',  // optional
});

console.log(vpn.connected);   // true
console.log(vpn.ip);          // '45.152.243.12'
console.log(vpn.expiresAt);   // '2026-05-12T...'

// Save this — needed to reconnect without paying again
console.log(vpn.sentinelMnemonic);

Security

Trust nothing. Verify everything.

The protocol is designed so that trust is unnecessary. Your keys, your tunnel, your traffic.

Agent controls the session

Agent's Sentinel private key signs MsgStartSession. Only the agent can start, extend, or end its VPN session.

Tunnel is direct

WireGuard/V2Ray handshake is agent-to-node. We never see the tunnel credentials, the encryption keys, or the traffic.

Payment is on-chain

USDC payment on Base or Solana. Fully auditable. No invoices, no billing systems, no chargebacks.

No lock-in

Agent's Sentinel key works with any operator, any plan, any node. Leave anytime. Use the same key with a different provider.
